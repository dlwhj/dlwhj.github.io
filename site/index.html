2024년 09월 21일 Using Variable Fonts in CSS

Variable fonts are an evolution of the OpenType font specification introduced in 2016. They combine the variants of a typeface into a single file, eliminating the need for separate files for each width, weight, or style.

In the context of CSS, using a variable font can condense the following: 

@font-face {
    font-family: "Nanum Square Neo";
    font-weight: 300;
    src: url("./assets/NanumSquareNeoOTF-Lt.otf") format("opentype");
}

@font-face {
    font-family: "Nanum Square Neo";
    font-weight: 400;
    src: url("./assets/NanumSquareNeoOTF-Rg.otf") format("opentype");
}

@font-face {
    font-family: "Nanum Square Neo";
    font-weight: 700;
    src: url("./assets/NanumSquareNeoOTF-Bd.otf") format("opentype");
}

@font-face {
    font-family: "Nanum Square Neo";
    font-weight: 800;
    src: url("./assets/NanumSquareNeoOTF-Eb.otf") format("opentype");
}

@font-face {
    font-family: "Nanum Square Neo";
    font-weight: 900;
    src: url("./assets/NanumSquareNeoOTF-Hv.otf") format("opentype");
}

into a single at-rule: 

@font-face {
    font-family: "Nanum Square Neo";
    src: url("./assets/fonts/nanum_square_neo_variable.woff2") format(woff2) tech(variations);
}

Using a variable font also provides access to a continuous range of weights, widths, and styles. Users are not limited to the few variants provided by the type designer.

For instance, I cannot apply a font weight of 600 using just the static fonts from the example above. However, with a variable font, I can specify any value between 100 and 900, including intermediate values like 650. The axis of variation describes the allowable range. These axes can be fine-tuned with the font-variation-settings property in CSS.
